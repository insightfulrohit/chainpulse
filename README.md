# ChainPulse – Blockchain News Dashboard

**ChainPulse** is a modern, real-time blockchain news dashboard designed to keep users updated with the latest happenings in the world of cryptocurrency. It aggregates trending news related to Bitcoin, Ethereum, DeFi, NFTs, Web3, and more — all in one place.

---

## Features

- 📰 **Live Crypto News Feed**  
  Get the latest updates from trusted sources like Cointelegraph, Coindesk, etc.

- **Real-Time Updates**  
  Stay informed with continuously refreshed blockchain news.

- 📊 **Category-Based Navigation**  
  Easily explore news across:
  - Bitcoin  
  - Ethereum  
  - DeFi  
  - NFT  
  - Web3  
  - Mining  
  - Security  

- 🌐 **Multi-Source Integration**  
  Aggregates news from multiple platforms into a unified interface.

- 🎯 **Featured News Highlight**  
  Displays the most important headline prominently.

- 🔍 **Search Functionality**  
  Quickly find relevant crypto news.

- 🌙 **Modern UI/UX**  
  Clean, dark-themed interface for better readability and focus.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Design:** Custom UI (Dark Theme Dashboard)  
- **Data Sources:** Crypto news APIs / RSS feeds *(customizable)*
